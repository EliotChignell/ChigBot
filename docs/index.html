ChigBot

A multi-purpose discord.js bot. InviteGitHub

Commands

Prefix: ch [command]

ch help

This serves as a starting point for new users. Use this command to find out the available commands.

ch ping

This is a simple command which tells the user the ping.

ch coinflip/coin/flipcoin

This command will simply flip a coin with the two possible outcomes of heads(true) and tails(false).

ch info

This command will tell the user links to the bot's Github page and my website.

ch encrypt [text to be encrypted]

This command returns your encrypted text and a key to decrypt it.

ch decrypt [key] [encrypted text]

This command returns your decrypted text.
